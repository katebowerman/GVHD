# GVHD MAG database

Methods and MAG description contained in [Continuous pre- and post-transplant exposure to a disease-associated gut microbiome promotes hyper-acute graft-versus-host disease in wild-type mice](https://www.tandfonline.com/doi/full/10.1080/19490976.2019.1705729)

Raw reads and genome assemblies available from NCBI under BioProject PRJNA544874

PROKKA annotated species representative MAGs (in GenBank format) can be downloaded [here](https://cloudstor.aarnet.edu.au/plus/s/zuZFpMwde3sgLGG)

PROKKA annotated non-dereplicated Enterobacteriaceae MAGs (in GenBank format) can be downloaded [here](https://cloudstor.aarnet.edu.au/plus/s/cdnAi0SQIYUNPnw)
